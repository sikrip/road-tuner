What is VVT tuner?

VVT tuner can help you tune the vvt settings on your variable valve timing engine (including Lift/Vtec activation) by using logged data with various VVT settings.

How should I create the log files?

You need to log wide open throttle runs with various VVT / VVTLi / VTec settings.

For example, for the 2zz Toyota engine with the Apexi PowerFC ECU, one should log wide open throttle runs with VVT setting set to 10, 20, 30, 40, 50 having the lift activation high enough (ex. 7500) starting from ~3K RPM and going up to ~7K(no lift activation) in order to acquire data for the low cam. Then put the lift activation low enough (say 3500 RPM) and log wide open throttle runs(from 3500 RPM to red line) with VVT set to 10, 20, 30, 40 and 50 in order to acquire data for the high cam.

Also make sure that you give each file name a descriptive name representing the VVT settings used. For example, give the log files names like low-cam-vvt-10.txt.

Finally, put all the log files in one folder and then load all of these files to this app.
